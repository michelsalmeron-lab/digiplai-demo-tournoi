Le Principe

Une animation mensuelle interactive sur comptoir :

  • Le Duel : Confrontation entre 2 E-cigarettes.
  • L'Arbitre : 1 e-liquide unique nommé L'Étalon LMDV.
  • Le Verdict : Le client vote sur tablette pour le meilleur rendu.
Note du Directeur : L'Étalon LMDV garantit une base de comparaison neutre et valorise la qualité de nos recettes.

Objectifs Commerciaux

1. Collecte de Leads : Acquisition d'emails qualifiés.

2. Expertise : Crédibiliser le conseil technique en boutique.

3. Engagement : Faire du client un acteur de la sélection produit.

4. Aide au choix : Orienter les ventes de matériel via le test réel.
